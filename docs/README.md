## Living vs. archival

These docs are **living** — update them when decisions change. The `mocks/` folder contains throwaway spec files (`SPEC.md`, `REFINED_SPEC.md`) used to brief design exploration; those are scratch and not authoritative. When a mock-spec choice is durable, promote it into the relevant doc here.
